# Uganda Electricity Distribution Loss Analysis

This repository contains a data-driven analysis of Uganda's electricity distribution losses using ERA regulatory data and GIS network information. The project is implemented in a Jupyter notebook that loads ERA Excel data, computes loss metrics, performs statistical tests and regression modeling, and visualizes the national and spatial distribution network.

## Project Contents

- `notebooks/TonyBlair_Ochieng_Uganda_final_project.ipynb`
  - Main analysis notebook for data loading, cleaning, EDA, regression, and GIS mapping.
- `data/raw/`
  - Raw ERA Excel files used for loss and network length analysis.
- `data/gis/`
  - GeoJSON files for Uganda distribution lines, transmission lines, and administrative boundaries.
- `data/fig9_interactive_map.html`
  - Interactive Folium map output generated by the notebook.

## Key Findings

- Distribution losses declined from around 20% in 2008 to about 14% in 2022.
- Rural networks show significantly higher loss rates than urban networks, with a mean gap of nearly 8 percentage points.
- The pre/post-2012 Bujagali period did not show a statistically significant difference in ERA loss rates.
- Regression analysis indicates that network length and rural status explain roughly 62% of the variance in loss rates.
- Spatial network density is highest in Central and Eastern regions, while West Nile, Northern, and South-Western districts remain more underserved.

## Data Requirements

Place the following files before running the notebook:

- `data/raw/Distribution_Database_Q3_2025-Updated.xlsm`
- `data/raw/Distribution_Statistics_Q4_2025_20260402.xlsm`
- `data/raw/Detailed-Statistics.xlsx` (optional)
- `data/gis/uganda_distribution-lines.geojson`
- `data/gis/ugandatransmissionlines2017.geojson`
- `data/gis/gadm41_UGA_1.json`

## Setup and Run

1. Create and activate a Python environment.

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

2. Open the notebook in Jupyter or VS Code and run the cells from top to bottom.

```bash
jupyter notebook notebooks/TonyBlair_Ochieng_Uganda_final_project.ipynb
```

## Notes

- The notebook uses the ERA `Distribution` and `Network_Length` sheets from the provided Excel files.
- GIS analysis reprojects line geometries to UTM for length calculations before mapping in WGS84.
